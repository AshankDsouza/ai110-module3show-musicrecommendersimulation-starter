# Assets

This folder stores project visuals used in documentation:

- System architecture exports (for example: `system-architecture.png`)
- CLI screenshots
- Experiment/profile output screenshots

## Mermaid.js workflow

1. Keep Mermaid source in `README.md` under **System Architecture**.
2. Open Mermaid Live Editor.
3. Paste the Mermaid diagram, then export PNG.
4. Save the exported image in this folder as `system-architecture.png`.

This keeps architecture docs reproducible and easy to review.
